# IT2010-MAD--ASSIGMENT03--Mobile-Application--Habit-Tracker-App
HabitTrackerApp is an Android wellness application built with Kotlin that helps users build healthy routines through one-tap habit tracking, mood journaling with emojis, hydration reminders, streaks, and progress insights, using offline-first storage with Shared Preferences for a smooth and reliable experience.
